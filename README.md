
# 🦸🏿‍♀️ CarolVerso – Classificador de Nível de Herói

##  Sobre o projeto

O **CarolVerso** é um projeto desenvolvido em JavaScript com o objetivo de classificar heróis de acordo com sua quantidade de experiência (XP).

A aplicação solicita ao usuário o nome do herói e o valor de XP, retornando automaticamente o nível correspondente entre as categorias:

- Ferro
- Bronze
- Prata
- Ouro
- Platina
- Ascendente
- Imortal
- Radiante

O projeto foi criado como prática de lógica de programação e conceitos básicos de JavaScript utilizando execução via terminal com Node.js.

##  Como executar o projeto

### 1. Instale o Node.js

Download oficial:

https://nodejs.org/

---

### 2. Clone ou baixe o projeto

```bash
git clone <url-do-repositorio>
````

---

### 3. Instale as dependências

No terminal, dentro da pasta do projeto:

```bash
npm install prompt-sync
```

---

### 4. Execute o projeto

```bash
node nivel_Heroi.js
```

---

##  Exemplo de execução

```bash
Bem vinde! Ao CarolVerso

Qual é o nome do seu herói?
Carol

Qual é o nível de experiência do seu herói? (Digite um número inteiro)
7500

O Herói de nome Carol está no nível de Platina
```

## 🏆 Tabela de níveis

| XP                    | Nível      |
| --------------------- | ---------- |
| Menor ou igual a 1000 | Ferro      |
| 1001 até 2000         | Bronze     |
| 2001 até 5000         | Prata      |
| 5001 até 7000         | Ouro       |
| 7001 até 8000         | Platina    |
| 8001 até 9000         | Ascendente |
| 9001 até 10000        | Imortal    |
| Maior que 10000       | Radiante   |
